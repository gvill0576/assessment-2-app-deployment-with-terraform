# Assessment II: AWS Serverless Application with AI Services

## Project Overview
Serverless application using AWS Lambda, API Gateway, DynamoDB, and AI services (Polly and Comprehend). Features a React frontend for user management and AI service interactions.

## Architecture
- **API Gateway:** REST API routing
- **Lambda Functions:** CRUD operations + AI services
- **DynamoDB:** User data storage
- **AI Services:** Amazon Polly (TTS) and Amazon Comprehend (Sentiment Analysis)
- **Frontend:** React application
